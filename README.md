# 🌌 Concept Engine — Physics Simulation Suite

## 🧠 Overview
Concept Engine is a Python-based interactive physics simulation project that visualizes and analyzes projectile motion using multiple approaches such as graphs, animations, and real-time control systems.

The goal of this project is to understand how physics behaves dynamically when parameters like angle and velocity are changed.

---

## 🎯 Features
- Static projectile motion graphs  
- Multi-angle comparison (30°, 45°, 60°)  
- Real-time interactive simulation using sliders  
- Frame-by-frame animation of motion  
- Multi-trajectory visualization  
- Comparative physics analysis system  

---

## 📁 Project Files

- `advanced_graph.py` → High quality static graphs  
- `basic_motion.py` → Basic physics implementation  
- `live_angle_changer.py` → Real-time angle control system  
- `normal_animation.py` → Single projectile animation  
- `multi_animation_graph.py` → Multiple trajectories animation  
- `projectile_motion_angles.py` → Angle-based study  
- `projectile_motion_comparison.py` → Comparison of cases  
- `real_time_engine_simulation.py` → Interactive physics engine  
- `simulation_motion.py` → General motion simulation system

## 📊 Physics Used

Projectile motion equations:

x = u cos(θ) t  
y = u sin(θ) t − ½ g t²  

Where:
- u = initial velocity  
- θ = launch angle  
- g = gravity  

---

## 📈 Key Observations
- 45° generally gives maximum range (ideal case)  
- Higher angles increase height but reduce range  
- Real-time interaction improves understanding of motion behavior
